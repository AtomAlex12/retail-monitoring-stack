# Changelog

## [0.2.1-beta] — 2026-02-27

### Исправлено
- **Remote write staleness:** точка оставалась зелёной 5+ минут после остановки Alloy. Добавлен фильтр `timestamp()` — точка считается онлайн только при данных за последние 2 мин (настраивается через `STALENESS_SEC`).

### Добавлено
- Документация `docs/remote-write-staleness.md`

---

## [0.2.0-beta] — 2025-02-25

### Добавлено
- Drill-down по точкам: клик открывает детали (ПК + роутер SNMP)
- API `/api/store/<name>` — метрики Windows и SNMP по точке
- API `/api/version` — версия экспортёра
- Файл `VERSION` — единый источник версии
- Бейдж версии в веб-интерфейсе

### Изменено
- Демо-страница обновлена под новый UI

---

## [0.1.0] — 2025-02-25

- Первый релиз
- Store Registry Exporter с веб-панелью
- Автообнаружение точек, хранение 30 дней
